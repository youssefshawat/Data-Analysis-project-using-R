# 📚 دليل شرح وظائف المشروع

## 🎯 نظرة عامة على المشروع

هذا المشروع يحلل بيانات أنظمة مشاركة الدراجات (Bikeshare) من 3 مدن أمريكية:
- **شيكاغو (Chicago)**
- **نيويورك (New York)**
- **واشنطن (Washington)**

---

## 📝 شرح كل وظيفة (Function)

### 1️⃣ `clean_dataset()` - تنظيف البيانات
```
الوظيفة: فحص وتنظيف وإصلاح القيم المفقودة في البيانات
```

**ماذا تفعل بالضبط:**
1. تفحص كل الأعمدة الموجودة
2. تحسب عدد القيم المفقودة (NA أو فارغة "")
3. تستبدل القيم المفقودة في الأعمدة الرقمية بـ **الوسيط (Median)**
4. تستبدل القيم المفقودة في الأعمدة النصية بـ **"Unknown"**
5. تحذف الصفوف التي ليس فيها وقت بداية الرحلة
6. تطبع تقرير مفصل عن التنظيف

**مثال:**
```
قبل: Gender = NA
بعد: Gender = "Unknown" (أو الوسيط للعمر)
```

---

### 2️⃣ `two_col_df()` - إنشاء جدول من عمودين
```
الوظيفة: إنشاء إطار بيانات (DataFrame) يتكون من عمودين
```

**ماذا تفعل بالضبط:**
1. تأخذ عمود بيانات واحد (مثل: مدة الرحلة)
2. تنشئ عمود جديد يحتوي على اسم المدينة
3. ترجع إطار بيانات بالعمودين

**مثال:**
```
الإدخال:  column = c(100, 200, 300)
         col_name = "Duration"
         city_name = "Chicago"

الناتج:  DataFrame مع عمودين:
         | Duration | City    |
         |----------|---------|
         | 100      | Chicago |
         | 200      | Chicago |
         | 300      | Chicago |
```

---

### 3️⃣ `get_month()` - استخراج الشهر
```
الوظيفة: استخراج رقم الشهر من تاريخ بداية الرحلة
```

**الناتج:** رقم الشهر من 01 إلى 12

**مثال:**
```
الإدخال: "2017-06-23 15:09:32"
الناتج:  "06" (يونيو)
```

---

### 4️⃣ `get_day()` - استخراج اليوم
```
الوظيفة: استخراج اليوم من الشهر (1-31)
```

**مثال:**
```
الإدخال: "2017-06-23 15:09:32"
الناتج:  "23"
```

---

### 5️⃣ `get_hour()` - استخراج الساعة
```
الوظيفة: استخراج الساعة من الوقت (0-23)
```

**مثال:**
```
الإدخال: "2017-06-23 15:09:32"
الناتج:  "15" (الساعة 3 مساءً)
```

---

### 6️⃣ `get_weekday()` - استخراج يوم الأسبوع
```
الظيفة: استخراج اسم يوم الأسبوع والترتيبه صح
```

**ماذا يختلف عن البسيط:**
- تأخذ التاريخ وتستخرج اسم اليوم
- ترتبه بالترتيب الصحيح (الإثنين → الأحد)

**مثال:**
```
الإدخال: "2017-06-23 15:09:32"
الناتج:  "Friday" (مرتب كـ Factor)
```

---

### 7️⃣ `count_table()` - جدول التكرار
```
الوظيفة: إنشاء جدول يوضح كم مرة تكررت كل قيمة
```

**مثال:**
```
الإدخال: c("Monday", "Monday", "Tuesday", "Monday")

الناتج:  | Var1    | Count |
         |---------|-------|
         | Monday  | 3     |
         | Tuesday | 1     |
```

---

### 8️⃣ `common_stations_summary()` - أشهر الرحلات
```
الوظيفة: إيجاد أكثر 5 رحلات تكراراً (من محطة لأخرى)
```

**ماذا تفعل:**
1. تاخذ محطة البداية والنهاية
2. تدمجهم في نص واحد مع سهم " -> "
3. تعد كم مرة تكررت كل رحلة
4. ترجع أفضل 5 رحلات

**مثال:**
```
الإدخال: Start="محطة A", End="محطة B"
الناتج:  "محطة A -> محطة B"

الجدول النهائي:
| المحطات                    | العدد |
|----------------------------|-------|
| محطة A -> محطة B          | 50    |
| محطة C -> محطة D          | 45    |
```

---

### 9️⃣ `check_missing_values()` - فحص القيم المفقودة
```
الوظيفة: فحص وتقرير القيم المفقودة بعد التنظيف
```

**الناتج:** تقرير مفصل بكل عمود فيه قيم مفقودة

---

## 🔄 تدفق العمل (Workflow)

```
1️⃣ تحميل البيانات
    ↓
2️⃣ تنظيف البيانات (clean_dataset)
    ↓
3️⃣ تحويل أنواع البيانات
    ↓
4️⃣ تحليل السؤال الأول
    ├── ملخص المدة
    ├── المدة حسب نوع المستخدم
    └── رسم بياني Boxplot
    ↓
5️⃣ تحليل السؤال الثاني
    ├── استخراج الوقت
    ├── جداول العد
    └── الرسوم البيانية
    ↓
6️⃣ تحليل السؤال الثالث
    ├── أشهر الرحلات
    └── الرسوم البيانية
    ↓
7️⃣ فحص الجودة + الملخص النهائي
```

---

## 📊 ملخص النتائج المتوقعة

### السؤال الأول: مدة الرحلة
| نوع المستخدم | المدة المتوسطة |
|--------------|----------------|
| مشترك (Subscriber) | 11-13 دقيقة |
| عميل (Customer) | 32-44 دقيقة |

### السؤال الثاني: أوقات الذروة
| الوقت | النمط |
|-------|--------|
| الشهر | يونيو هو الأعلى |
| اليوم | weekend أقل استخدام |
| الساعة | ذروة صباحية + مسائية |

### السؤال الثالث: أشهر الرحلات
- المناطق السياحية المركزية
- رحلات الذهاب والعودة لنفس المحطة
